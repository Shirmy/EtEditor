## 这是什么

EtEditor 是一个把 EPUB 和 TXT 电子书的常见整理工作集中到一起的编辑器。把书导入进来,可以整理目录、批量替换文本、抓取书籍信息、生成封面,再导出回 EPUB 或 TXT。

## 主要功能

**通用**
- EPUB / TXT 双格式编辑
- 大文本窗口化加载,边缘自动扩张
- 文本替换(支持预设、批量、范围选择、预览)
- 自动化执行链(把多个步骤串起来一键执行)
- 导入 / 导出配置

**TXT**
- 目录识别与规则映射
- 章节增删、移动、排序
- 全文整理与格式化
- 长短章自动计算
- 书名过滤

**EPUB**
- 分卷、分章、插入章节
- 插入图片、生成封面
- 信息抓取(辅助补全书籍信息)
- ncx / opf 缩进输出

## 技术栈

- Kotlin

## 关于抓取功能

应用里的"信息抓取""插入章节"等功能用于辅助编辑,需要用户**自行提供目标网站的登录凭证(Cookie)**,应用本身不内置任何账号。

请在使用时遵守目标网站的服务条款,仅用于个人合法用途,不要用于侵权或商业搬运。因使用本工具抓取内容产生的任何责任由使用者自行承担。
